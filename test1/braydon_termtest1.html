HTML

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markuplanguage for creating Webpages
  • HTML describes the structure of a Webpage
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements lable pieces of content such as "this heading", "this is a link", etx.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS descibes how HTML elements are to be displayed on screen, paper; or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Python

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991. It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.

PHP

  • PHP is a server scripting language, and a powerful tool for making dynamic and interactive Web pages.
  • PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP.
  • PHP 7 is the latest stable release.
